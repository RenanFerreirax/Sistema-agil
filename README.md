🐍 Smoking Snake

Tradição no comércio, inovação em tecnologia

📖 Sobre o Projeto

A Smoking Snake é uma empresa brasileira fundada em 1983 por Juvenal Sonta, em uma região periférica de São Paulo. O negócio começou como um pequeno comércio voltado à venda de cigarros e charutos, crescendo ao longo dos anos até se tornar uma empresa com atuação também no setor de tecnologia.

Hoje, a empresa combina sua tradição comercial com soluções digitais modernas.

✨ Funcionalidades

Tela de Login: Formulário com validação de campos (email e senha).

Dashboard Protegido: Página acessível apenas para usuários autenticados.

Logout: Funcionalidade para encerrar a sessão do usuário.
