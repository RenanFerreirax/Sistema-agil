# 🐍 Smoking Snake

**Tradição no comércio, inovação em tecnologia**

---

## 📖 Sobre o Projeto

A **Smoking Snake** é uma empresa brasileira fundada em 1983 por Juvenal Sonta, em uma região periférica de São Paulo.

O negócio teve início como um pequeno comércio voltado à venda de cigarros e charutos. Com o passar dos anos, a empresa cresceu e expandiu sua atuação, passando a integrar também soluções no setor de tecnologia.

Hoje, a **Smoking Snake** combina sua sólida tradição comercial com ferramentas digitais modernas, oferecendo uma experiência mais eficiente e inovadora.

---

## ✨ Funcionalidades

- 🔐 **Tela de Login**
  - Formulário com validação de campos (e-mail e senha)

- 📊 **Dashboard Protegido**
  - Acesso restrito apenas para usuários autenticados

- 🚪 **Logout**
  - Encerramento seguro da sessão do usuário

---

## 🚀 Tecnologias Utilizadas

- Node.js
- Express

---

## 📌 Objetivo

Este projeto tem como objetivo demonstrar a implementação de um sistema básico de autenticação com proteção de rotas, unindo conceitos de backend com uma aplicação prática.
